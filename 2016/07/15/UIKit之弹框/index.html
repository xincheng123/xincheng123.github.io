UIKit之弹框

在iOS开发中系统弹框有三种方式

iOS8 之前用        UIAlertView(中间弹出)       UIActionSheet(下放弹出)

iOS8 之后用        UIAlertController(可定义中间或下边弹出)

一:用UIAlertView创建弹窗(从中间弹出的弹框)

  1. 创建对象:(超过两个按钮就垂直排放,这里定义按钮的顺序就是按钮的下标)
    注意

    弹窗类型
UIAlertViewStyleDefault = 0,
UIAlertViewStyleSecureTextInput,
UIAlertViewStylePlainTextInput, //有一个文本输入框
UIAlertViewStyleLoginAndPasswordInput

//通过UIAlertView创建的弹窗从中间弹出****
//1.创建弹窗
UIAlertView * alertView = [[UIAlertView alloc] initWithTitle:@"提示弹窗" message:@"这是一个提示弹窗" delegate:self cancelButtonTitle:@"取消为第0个" otherButtonTitles:@"确定1",@"选择2",@"第三个按钮3", nil];

2.修改弹窗类型     
//带一个文本框的弹窗

alertView.alertViewStyle = UIAlertViewStylePlainTextInput;

3.获取弹窗中的textField(用当前alertVIew调用)
UITextField * textField = [alertView textFieldAtIndex:0];

//2.显示弹窗
[alertView show];
//3.实现代理方法,为弹窗按钮添加点击事件

  1. 两个按钮时,取消在左边,另一个按钮在右边,取消按钮下标为0

  2. 两个以上的按钮垂直排列:

  3. 点击一个按钮后要执行的的操作写在这个代理方法里

    // 实现UIAlertView的代理方法<需要代理遵守UIAlertViewDelegate 协议>
- (void)alertView:(UIAlertView *)alertView clickedButtonAtIndex:        (NSInteger)buttonIndex
{
NSLog(@"%ld",buttonIndex); // 取消 0  确定 1
if (buttonIndex == 1) { // 表示用户点击了确定
// 进入下一个游戏环节
self.index = -1;
// 调用下一题点击方法
[self nextQuestionBtnClick];
}

    1. 补充:

      修来弹出对话框的样式 alertView.alertViewStyle = UIAlertViewStylePlainTextInput;

      [alertView textFieldAtIndex:0] → [alertView textFieldAtIndex:0].text = hero.name;

二:用UIActionSheet创建弹窗(从下方弹出的弹框)

  1. 创建弹窗,显示弹窗

    //通过ActionSheet创建的弹框从下边弹出
//1.创建弹窗
UIActionSheet * actionSheet = [[UIActionSheet alloc] initWithTitle:@"这是一个底部弹窗" delegate:self cancelButtonTitle:@"取消按钮最后一个" destructiveButtonTitle:@"删除按钮为第0个" otherButtonTitles:@"添加1",@"计算2", nil];
//2.显示弹出
[actionSheet showInView:self.view];
//3,实现代理,为弹窗按钮添加点击事件

  2. 按钮的位置排布

三:用UIAlertController创建弹窗(通过含nib的init方法可以自定义弹框,不需要遵守协议)

  1. 创建弹框,添加按钮,显示弹框

    //方法三:通过UIAlertController创建一个从中间弹出的弹框
//    UIAlertControllerStyleActionSheet = 0,    //从底部弹出
//    UIAlertControllerStyleAlert               //从中间弹出
//1.创建一个没有按钮的弹窗

//1.创建一个空弹窗
UIAlertController * alertController  = [UIAlertController alertControllerWithTitle:@"这是一个弹窗" message:@"llllllllllll" preferredStyle:UIAlertControllerStyleActionSheet];

//2.创第一个按钮
UIAlertAction * alertAction1 = [UIAlertAction actionWithTitle:@"取消按钮" style:UIAlertActionStyleCancel handler:^(UIAlertAction * _Nonnull action) {
NSLog(@"你点击了取消按钮");
}];

//创第二个按钮

UIAlertAction * alertAction2 = [UIAlertAction actionWithTitle:@"格式化" style:UIAlertActionStyleDestructive handler:^(UIAlertAction * _Nonnull action) {
NSLog(@"你点击了格式化按钮");
}];
//创第三个按钮

UIAlertAction * alertAction3 = [UIAlertAction actionWithTitle:@"添加按钮" style:UIAlertActionStyleDefault handler:^(UIAlertAction * _Nonnull action) {
NSLog(@"你点击了添加按钮");
}];
//3.把按钮添加到弹弹窗

//把按钮添加到弹窗中
[alertController addAction:alertAction1];
[alertController addAction:alertAction2];
[alertController addAction:alertAction3];

//4.让控制器动画调用弹窗
//显示弹窗(让控制器调用刚刚创建的弹窗  ,表示使用该弹窗,completion后面接弹窗一弹出马上执行的操作)
[self presentViewController:alertController animated:YES completion:nil];

  2. 在弹框中添加文本输入框

    //弹窗中添加一个textfield

[alert addTextFieldWithConfigurationHandler:^(UITextField * _Nonnull textField) {
//可做一些操作,比喻打印文本输入框中的值
 NSLog(@"%@",textField.text);
}];

  3. 获取当前弹框的文本输入框的文字

    NSString *txt = alert.textFields.lastObject.text;
