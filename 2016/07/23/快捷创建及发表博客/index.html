快捷创建博客及发表博客

1.打开终端,cd到hexo目录下

2.进入hexo目录后,输入以下目录,创建博客文件,文件是.md后缀的

hexo new   博客名称

比如我想创建一个名为集成多说评论模块的博客,代码如下

hexo  new  集成多说评论模块

3.现在就可以到/hexo/source/_posts/目录下找到你刚刚创建的.md后缀的文件,而且自动生成了一个同名的文件夹,用于放你该篇博客中用到的图片资源

4.双击打开.md文件开始编辑博客,编辑完成后保存,然后回到终端,输入以下命令

  1. 生成静态网页

    hexo generate

  2. 打开hexo服务

    hexo server

    打开hexo服务后就可以在浏览器中通过 localhost:4000 本地预览以下你发布的博客,此时别人还没法看到,因为还需要把博客部署到github上

  3. 关闭hexo服务

    Control  + C    //关闭hexo服务

  4. 部署到github

    hexo deploy

部署到github上时,如果你需要部署的图片资源比较多,可能需要等一会才能上传完成,等待上传完成,就可以通过你自己的博客地址查看你的博客了
