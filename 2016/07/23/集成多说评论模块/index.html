1.替换通用代码

找到自己使用的hexo\hemes的文件夹里面的comment.ejs的文件，例如我的主题是名称叫vno，
那么comment.ejs文件就位于的路径是hexo\themes\vno\layout_partial\comment.ejs，打开文件,并将里面的代码全部删除,替换为以下代码。
(.ejs文件如果你没有工具打开,可以修改后缀名称为.js,打开.js后修改完毕保存时再改回.ejs文件即可)

2.修改通用代码

将data-thread-key="请将此处替换成文章在你的站点中的ID"替换成data-thread-key=”<%= page.path %>”

将data-title=”请替换成文章的标题”替换成data-title=”<%= page.title %>”

将data-url=”请替换成文章的网址”替换成 data-url=”<%= page.permalink %>”
修改完后大致如下：

<% if (page.comments){ %>  
<section id="comment">  
<!-- 多说评论框 start -->
<div class="ds-thread" data-thread-key="<%= page.path %>" data-title="<%= page.title %>" data-url="<%= page.permalink %>"></div>
<!-- 多说评论框 end -->
<!-- 多说公共JS代码 start (一个网页只需插入一次) -->
<script type="text/javascript">
var duoshuoQuery = {short_name:"xinching"};
(function() {
var ds = document.createElement('script');
ds.type = 'text/javascript';ds.async = true;
ds.src = (document.location.protocol == 'https:' ? 'https:' : 'http:') + '//static.duoshuo.com/embed.js';
ds.charset = 'UTF-8';
(document.getElementsByTagName('head')[0] 
 || document.getElementsByTagName('body')[0]).appendChild(ds);
})();
</script>
<!-- 多说公共JS代码 end -->  
</section>  
<% } %>

修改完成后保存为.ejs后缀的文件
