hexo博客集成Disqus评论模块

这里先说一下为什么不久前集成了多说评论功能,现在又集成这个Disqus评论模块,主要原因是有个小伙伴告诉我多说评论模块会自动把我们个人博客的文章转载上传到他们的服务器上,如果真是这样就太不安全了有木有!,所以就集成了这个Disqus评论模块!(不过有一点比较坑,就是Disqus评论模块只有翻墙才显示,不翻墙无法成功加载Disqus评论模块)

首先是登陆Disqus官网注册账号,为您的网站配置Disqus

网址:

注册然后登陆:

点击图中的Home

点击Add Disqus To Site选项

把界面拉到最下面,点击GET STARTED按钮

出现如下创键界面,按图中要求输入内容,点击Create Site按钮创键

点击跳过这个界面

点击最上面的Settings按钮

把你刚刚创键的Disqus网站和你的博客关联起来

至此,Disqus服务器上的网站创建完毕也关联完毕,其中需要记住,你的Disqus网站的short_name就是xincheng,你的博客的short_name是xincheng123,要区分这两者,后面会用到的是你Disqus网站的shor_name.

现在需要在本地修改配置文件,让本地的hexo网站关联Disqus网站

  1. 在你的本地hexo目录下找到themes文件夹,找到 你当前应用的主题文件夹,我用的是vno主题,打开vno主题文件夹,找到comments.ejs文件

用下面的代码直接覆盖掉comments.ejs文件中的内容,然后修改disqus_shortname = ''单引号中的值为你Disqus网站的short_name

<% if (page.comments){ %>  
<section id="comment">  
<!-- Disqus评论框 start -->
<section id="comments">
<div id="disqus_thread"></div>
<script type="text/javascript">
/* * * CONFIGURATION VARIABLES: EDIT BEFORE PASTING INTO YOUR WEBPAGE * * */
var disqus_shortname = '这里替换成你注册时Disqus网站的short_name'; // required: replace example with your forum shortname

/* * * DON'T EDIT BELOW THIS LINE * * */
(function() {
  var dsq = document.createElement('script'); dsq.type = 'text/javascript'; dsq.async = true;
  dsq.src = '//' + disqus_shortname + '.disqus.com/embed.js';
  (document.getElementsByTagName('head')[0] || document.getElementsByTagName('body')[0]).appendChild(dsq);
})();
</script>
<noscript>Please enable JavaScript to view the <a href="https://disqus.com/?ref_noscript">comments powered by Disqus.</a></noscript>
</section>
<!-- Disqus公共JS代码 end -->  
</section>  
<% } %>

设置short_name以后的代码如图:

然后打开终端:cd到hexo目录下,键入
hexo s
命令,用浏览器本地查看效果

成功后直接提交到github上即可,在终端输入下面命令提交:

hexo     generate
hexo      deploy

现在既可以去自己的博客看看刚刚集成的Disqus留言模块了.以后就可以和小伙伴快乐的评论了!
